# Product Requirements Document (PRD): The Hezron Terminal (V1.0)

## 1. Executive Summary
This document defines the requirements for a high-performance, dual-mode portfolio website for **Yonathan Hezron**. [cite_start]The platform serves as a "Command Center" that integrates **Network Engineering** (MikroTik MTCRE/MTCNA) with **Full-stack Development** and **Marketing Strategy**[cite: 2, 6, 15, 16]. [cite_start]The objective is to demonstrate "Industrial Awareness" through a UI that balances high-tech aesthetics with corporate reliability[cite: 8].

## 2. Design Language & Visual Identity
The website will utilize a **Hybrid Theme Architecture**, allowing users to toggle between two distinct professional personas.

### 2.1. Main Color Palettes
* **The Dark Mode (Cyber-Industrial):**
    * **Background:** Deep Obsidian (#0D0D0D) with subtle grid-line textures.
    * **Accents:** Electric Cyan and "Glitch" Magenta for high-action elements like "Instantly" or "Live Activity".
    * **Reference:** This mode mirrors the **GamerZone** and **Synapse.AI** interfaces.
* **The Light Mode (Enterprise-Minimalist):**
    * **Background:** Stark Paper-White (#FAFAFA) with soft grey elevation (#E2E8F0).
    * **Accents:** Professional Deep Cobalt (#1E293B) and Emerald Green for "System Online" status indicators.
    * **Typography:** Monospace fonts for technical data (JetBrains Mono) and clean Sans-Serif for business prose (Inter/Roboto).

### 2.2. Visual Assets Integration
* **Pre-loader Animation:** Use the circular "Calibrating Nav Systems" graphic from **1-login.png** to mask initial data fetching.
* **Hero Imagery:** Use the "Sovereign Intelligence" layout from **01_hero_section.jpg** as the primary template for the landing experience.
* **Status Indicators:** Real-time badges for "Total Transactions" or "Active Users" to simulate a live-production environment.

## 3. Functional Requirements

### 3.1. The "Nexus" (Hero Module)
* [cite_start]**Dynamic Identity Header:** A scrolling text-switcher displaying: "Network Engineer," "Full-stack Developer," and "Marketing Specialist"[cite: 2].
* **Status Badge:** A "System Operational" indicator linked to a "Jakarta Node-01" label, reflecting your location in Batam/Kepulauan Riau.
* **CTA (Call to Action):** Primary button "Jalankan Demo" (Run Demo) and secondary "Lihat Infrastruktur" (View Infrastructure).

### 3.2. "The Arsenal" (Project Showcase)
Inspired by the **GamerZone** catalog, this section will host your previous works.
* **Search & Filter Engine:** Users can filter projects by technical tags like "React.js," "Node.js," "PHP," "Python," "MikroTik," or "SQL".
* **Interactive Cards:** Each project card must display a "Featured" badge and a "Get It Now" direct link.
* **Empty State:** If a search yields no results, display the "No Games Found" icon and a "Reset Filters" prompt.

### 3.3. "Sovereign Infrastructure" (Technical Skillset)
Based on the **Synapse.AI** tiers, display your technical proficiencies as "Services":
* [cite_start]**Tier 1: Smart Entry Build:** Focus on MikroTik routing (MTCNA), OSPF, and basic troubleshooting[cite: 16, 17, 33].
* **Tier 2: Prosumer Powerhouse:** Focus on MTCRE certification, BGP routing, and RAG Pipeline setups.
* **Tier 3: Enterprise Cluster:** Focus on Full-stack React/Node.js, PostgreSQL databases, and API integration.

### 3.4. "Intelligence Stream" (Experience & News)
A vertically scrolling feed styled after the **Intelligence** dashboard:
* **Category Tags:** "Patch Notes" (Project Updates), "Lore & Narrative" (Career History), and "Esports" (Competitive achievements or marketing wins).
* **Live Activity:** A notification bar showing "7 Transactions Successfully Processed!" or similar achievements in your HMSI or PMK organization roles.

## 5. Technical Specifications
* [cite_start]**Framework:** React.js for the interface with Tailwind CSS for rapid dark/light theme switching[cite: 34].
* **Backend:** Node.js managing a "Secure Vault" for document analysis and file management.
* **Security:** Implementation of "256-bit SSL encryption" and "Firewall Security" protocols to mirror the "Secure Transactions" requirement.
* **Data Visualization:** Integration of "Analisis ROI" charts using Recharts or D3.js to visualize code efficiency or network performance.
